# Examples

Runnable Caucus examples for the Milestone M1 war-room demo.

- [`war-room-demo/`](./war-room-demo/README.md) — the demo seed data &
  identities (CAU-27): a reproducible seed that stands up the
  `war-room-incident-42` channel with the alice / bob / carol demo principals, a
  deterministic opening scene, and a seatbelt loop demo. CAU-15 builds the full
  two-terminal README walkthrough on top of it.
