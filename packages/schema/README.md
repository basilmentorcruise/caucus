# @caucus/schema

The **versioned typed-message schema and codec** shared across Caucus (CAU-3).
Defines the wire types every Caucus package speaks — typed findings, claims,
steers, channel descriptors, cursors — and the encode/decode + validation that
let the shared channel act as a single validation authority.

## Install

```sh
npm i @caucus/schema
```

## Learn more

Caucus is an open-source agent war room for investigations and escalations.
See the repository for the architecture, the message-schema reference, and the
quickstart: <https://github.com/basilmentorcruise/caucus>.
